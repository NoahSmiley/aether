# Aether

Custom Jellyfin client for Apple TV, built with SwiftUI.

Part of the [Athion](https://athion.me) ecosystem alongside Liminal, Flux, and OpenDock.

## Status

Research & planning phase.

## Tech Stack

- **Platform**: tvOS (Apple TV)
- **UI**: SwiftUI
- **Video**: AVKit / AVPlayer (native HLS support)
- **Backend**: Jellyfin REST API
- **Language**: Swift
